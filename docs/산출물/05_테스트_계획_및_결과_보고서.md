# 테스트 계획 및 결과 보고서

## 1. 테스트 개요

### 1.1 테스트 목적

요구사항 정의서(01_요구사항_정의서.md)에 정의된 기능 요구사항을 기반으로 ZIPFIT 애플리케이션의 기능, 성능, 안정성을 검증합니다.

### 1.2 테스트 범위

- **기능 요구사항 테스트**: 요구사항 ID 기반 기능 검증
- **API 테스트**: REST API 엔드포인트 검증
- **LangGraph 테스트**: 챗봇 의도 분류 및 응답 검증
- **비기능 요구사항 테스트**: 성능, 사용성, 보안, 안정성 검증
- **통합 테스트**: 프론트엔드-백엔드 연동 검증

### 1.3 테스트 환경

| 항목 | 내용 |
|------|------|
| 테스트 서버 |  http://{application_server_ip}:8080/ (AWS Lightsail) |
| 데이터베이스 | PostgreSQL 14.20 + pgvector (AWS Lightsail) |
| 테스트 데이터 | 실제 LH 공고 데이터 29건 |
| 테스트 일시 | 2025-12-17 |
